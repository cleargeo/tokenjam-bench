## Honesty Discipline

Every document in this repo follows TokenJam's honesty discipline:

- Savings are "estimated recoverable," never "saved"
- Quality claims are pass-rate measurements on specific suites, never general "equivalence"
- Small samples are flagged as insufficient for significance
- Mock runs are flagged as illustrative
